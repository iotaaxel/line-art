## Creating Your Own

1. Start interactive mode: `python main.py`
2. Set canvas size (e.g., `30 30`)
3. Enter coordinates: `x1,y1 x2,y2 x3,y3 x4,y4 ...`
4. Type `show` to preview
5. Type `done` when finished

## Tips

- Use even numbers of coordinates (pairs)
- Odd numbers will ignore the last coordinate
- Lines are automatically clipped to canvas bounds
- You can enter multiple lines in one input!
